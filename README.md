# currency-converter-app
This is a currency converter app which converts any currency to any other currency, it has supports 30+ currencies

## Features
1. It can convert any currency to any other currency you want.
2. Sources are reliable and trusted.
3. Rates are alomst accurate and latest.
4. If you are going for a foreign trip, you can use it confidently to get the true and accurate rates.
